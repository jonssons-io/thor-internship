HTML & CSS

Varje hemsida är uppbyggd av HTML, HyperText Markup Language, och CSS (Cascading Style Sheets). HTML är webbsidans skelett medan CSS är utseendet på sidan, t.ex. färg, position, mellanrum och teckensnitt.

HTML - Hemsidans skelett

  1. Titta på videorna i denna spellista: HTML & CSS - Grunder
  2. Skapa ett HTML dokument i VS Code som du döper till index.html
  3. Lägg in alla nödvändiga grundtaggar som du lärt dig i första videon.
    • <!DOCTYPE html> som berättar att dokumentet är av typen HTML5
    • <html> som berättar för webbläsaren var html koden börjar och slutar. Glöm inte språkattributet lang och sluttaggen.
    • <head> som innehåller information som oftast inte syns på hemsidan, men som ändå behövs. Glöm inte sluttaggen.
    • <meta> som är en tagg för att markera meta-information, t.ex. vilken teckenuppsättning man använder (attributet charset).
    • <title> som innehåller det som står längst upp i tabben, det vill säga namnet på sidan. Glöm inte sluttaggen.
    • <body> som markerar innehållet på sidan, det vill säga det som besökaren kommer att se. Glöm inte sluttaggen.
  4. Testa lägga in olika element innanför <body> och se vad som händer. Lägg till minst 3 olika element.

CSS - Hemsidans smink

  1. Repetera videorna om CSS i spellistan om du känner att du behöver det.
  2. Skapa ett CSS dokument på samma nivå som ditt index.html dokument, och döp det till style.css.
  3. Importera CSS dokumentet i din index.html. Importen ska ligga längst ned inuti <head>.
  4. Ändra bakgrundsfärgen i din övningssida genom att lägga till attributet style på body taggen.
  5. Ändra teckensnitt (font-family) på alla dina h1-element till valfritt teckensnitt.
  6. Ändra teckensnitt på alla andra textelement. Tänk på att den CSS-regeln som är längst ned i dokumentet skriver över alla regler ovanför.
  7. Skapa ett textelement i index.html och lägg till attributet class. Använd klassen för att ändra färg på textelementet.
