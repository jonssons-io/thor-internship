Thors praktikhemsida

Här hittar du alla instruktioner och en del resurser där du kan läsa på om HTML och CSS. Använd navigationen högst upp på sidan för att komma igång.

  1. Börja med setup. Där finns instruktioner om hur du installerar kodeditorn och andra verktyg.
  2. Första övningarna ligger under HTML & CSS där du får sätta igång och öva på att skapa en sida med HTML och sedan använda CSS för att ändra färger och form.
  3. När du är klar med HTML & CSS, går du till projekt där du får använda dig av det du lärt dig tidigare, och skapa din egen hemsida.
